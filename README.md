# Site da Editora Gnosis Carajás

Site estático pronto para publicação no GitHub Pages.

## Estrutura

- `index.html`: página principal do site.
- `assets/css/styles.css`: estilos visuais.
- `assets/js/data.js`: dados do catálogo.
- `assets/js/script.js`: busca, filtros e renderização do catálogo.
- `assets/covers/`: capas extraídas dos PDFs.
- `pdfs/`: arquivos PDF disponíveis para download no catálogo.

## Publicação no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie todos os arquivos e pastas deste pacote para a raiz do repositório.
3. Acesse Settings > Pages.
4. Em Source, selecione Deploy from a branch.
5. Escolha a branch `main` e a pasta `/ (root)`.
6. Salve e aguarde a publicação.

## Observação sobre formulários

Os formulários de contato e normas para autores são visuais. Para funcionar no GitHub Pages, será necessário integrar FormSubmit, Google Forms ou outro serviço externo.
