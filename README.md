# Editora Gnosis Carajás

Site estático preparado para GitHub Pages.

## Estrutura simplificada

Arquivos na raiz:
- index.html
- sobre.html
- catalogo.html
- normas.html
- contato.html
- styles.css
- data.js
- script.js
- logo.png
- favicon.ico
- robots.txt
- sitemap.xml
- EXEMPLO_INSERIR_NOVO_LIVRO.txt

Pastas:
- covers/  (capas das obras)
- pdfs/    (arquivos PDF das obras)

## Como editar o catálogo

Edite o arquivo data.js. Ele possui comentários e um modelo pronto para copiar e colar.

## Destaques da página inicial

No final do arquivo data.js, altere a lista DESTAQUES_INICIO.
